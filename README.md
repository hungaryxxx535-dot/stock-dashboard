# 非哥股票作战台

手机优先的个人股票持仓观察、行情快照、风控提醒与交易复盘工具。

当前版本口径：持仓清单由手动维护；A股盘中页已接入 AKShare/公开行情；美股收盘页已接入 Twelve Data 收盘价，并在接口失败时回退到本地占位数据。当前版本不连接券商账户，不自动下单，不承诺收益。

## 技术栈

- Next.js App Router 项目结构
- React 页面组件
- Tailwind CSS 样式约定
- shadcn/ui 风格本地组件
- lucide-react
- framer-motion

## 本地运行

```bash
npm install
npm run dev
```

默认端口：`3000`。

启动后访问：<http://localhost:3000>

## 功能范围

- 首页总览：展示手动维护的 A股/美股持仓结构、机动现金、总仓位、风险灯和复核清单。
- A股盘中：通过行情服务读取公开行情；接口不可用时显示本地持仓快照回退。
- 美股收盘：通过 Twelve Data 读取核心美股收盘价；接口不可用时显示本地回退数据。
- 风险提醒：基于当前持仓结构提示仓位、集中度、单票和现金垫风险。
- 操作线：记录盘前、盘中、收盘后的纪律线、观察记录和更新清单。
- 参数设置：手动调整现金、汇率等参数，用于总览和风险页估算。

## 数据口径

- 持仓数量、成本价、备注：来自手动维护的 `src/data/portfolio.ts`。
- A股价格：优先看 `/a-live` 页面接入的盘中行情。
- 美股价格：优先看 `/us-close` 页面接入的收盘价。
- 首页持仓总览：用于结构观察和复盘，不等同于券商账户实时同步。
- 交易动作：本项目只提供观察和复盘辅助，不提供自动交易或自动下单。

## 日常更新方式

不需要每天盘后上传截图。只有发生买入、卖出、加仓、减仓，或者成本价发生变化时，才需要更新持仓底表。没有交易变动时，只需要刷新行情页，系统会根据持仓数量、成本价和最新行情估算市值与浮盈亏。
